## 项目概述

新能源汽车销售小程序是一款面向插电混动车型销售的个人工具应用。小程序为销售人员提供了一个专业的数字化展示平台，帮助客户在线了解车型信息、计算落地价格、预约试驾，同时收集高质量的销售线索。

**目标用户**：新能源汽车销售顾问及其客户

**核心价值**：在微信生态内完成从车型展示到客户留资的完整链路，无需跳出微信。

## 车型展示

小程序覆盖 4 款插电混动车型，每款车型都有独立的详情页面：

### 旗舰 SUV

定位高端商务市场：

- 搭载 1.5T + 电机插混系统，综合功率 380kW
- 纯电续航 200km+（CLTC 工况）
- 6 座布局，二排独立航空座椅
- 智能驾驶辅助系统，支持高速 NOA

### 大型家庭 SUV

主打大空间和舒适性：

- 1.5T 插混系统，综合续航 1200km+
- 6/7 座灵活布局
- 二排小桌板 + 车载冰箱
- 全系标配 L2+ 级驾驶辅助

### 中型运动 SUV

兼顾运动与家用：

- 2.0T 插混系统，百公里加速 5.7 秒
- 纯电续航 155km（WLTC 工况）
- 运动化底盘调校
- 14.6 英寸中控大屏

### 紧凑型都市 SUV

面向年轻消费群体：

- 1.5T 插混系统，油耗低至 0.8L/100km
- 纯电续航 175km（WLTC 工况）
- 潮流设计语言
- 高通骁龙 8155 座舱芯片

每款车型的详情页包含：高清外观/内饰图集、核心参数表、车型亮点卡片、同级对比表。

## 落地价计算器

这是小程序最核心的功能之一，帮助客户快速了解购车的真实成本。

### 计算项

- **裸车价**：官方指导价 - 终端优惠
- **购置税**：插电混动车型享受购置税减免政策（2026 年减半征收）
- **保险费用**：交强险 + 商业险（车损、三者 200 万、座位险）
- **上牌费用**：包含牌照费和检测费
- **金融服务**：如选择贷款，计算首付比例、月供金额、总利息

### 使用流程

1. 选择车型和配置版本
2. 输入终端优惠金额（销售顾问可现场填写）
3. 选择全款或贷款方案
4. 系统自动计算各项费用并生成落地价明细表
5. 一键生成图片，可保存或分享给客户

## 客户线索收集

小程序内置客户留资表单，在多个场景触达客户：

### 触达场景

- **车型详情页底部**：「对该车型感兴趣？留下联系方式，专属顾问为您服务」
- **落地价计算完成后**：「保存报价方案，顾问将为您争取更大优惠」
- **首页弹窗**：首次进入小程序时的欢迎引导

### 表单字段

- 姓名（必填）
- 手机号（必填，带格式校验）
- 意向车型（下拉选择）
- 预算范围（下拉选择）
- 购车时间（下拉选择：1 周内 / 1 个月内 / 3 个月内 / 暂不确定）
- 备注信息（选填）

### 线索管理

- 所有线索存储在微信云开发数据库中
- 销售顾问可在小程序后台查看和管理线索
- 支持按时间、车型、状态筛选
- 线索状态：新建 → 已联系 → 已到店 → 已成交 / 已流失

## 销售顾问电子名片

每位销售顾问拥有专属的电子名片页面，包含：

- 个人照片和姓名
- 所属经销商和门店地址
- 联系电话和微信号
- 从业年限和专业领域
- 个人二维码（可保存到相册分享）

名片支持一键拨打电话、一键添加微信、一键导航到店。客户通过名片页面留资的线索会自动关联到该顾问名下。

## 技术架构

| 层级 | 技术 |
|------|------|
| 前端 | WXML / WXSS / JavaScript |
| 后端 | 微信云开发（云函数 + 云数据库） |
| 存储 | 微信云存储（车型图片） |
| 数据库 | 微信云开发数据库（线索、顾问信息） |
| 接口 | 微信开放能力（登录、分享、订阅消息） |

### 云函数

- `submitLead`：处理线索提交，发送订阅消息通知顾问
- `getAdvisor`：获取顾问信息和名片数据
- `getModels`：获取车型配置和价格数据
- `generateQuote`：生成落地价计算结果

### 页面结构

```
├── pages/
│   ├── index/          # 首页（车型列表）
│   ├── model/          # 车型详情
│   ├── calculator/     # 落地价计算器
│   ├── advisor/        # 顾问名片
│   └── leads/          # 线索管理（顾问端）
```
